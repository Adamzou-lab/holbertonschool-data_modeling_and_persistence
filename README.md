# Holberton Data Modeling and Persistence

Module Holberton School sur la modélisation de données et la persistance en SQL.

## Projets

| Dossier | Contenu |
|---|---|
| `sql_crud` | Opérations de base sur une base relationnelle : création de tables, insertion, lecture, mise à jour et suppression |
| `sql_join` | Jointures, sous-requêtes et regroupements pour interroger plusieurs tables |

## Prérequis

- MySQL 8 ou MariaDB

## Utilisation

```bash
cat fichier.sql | mysql -u root -p nom_de_la_base
```
